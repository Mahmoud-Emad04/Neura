🔐 Authentication

POST /auth/login Public Authenticate & get JWT token

Description

Authenticates a user with username/email and password. Returns a JWT access token and a refresh token.

Request Body

{
  "userNameOrEmail": "john@example.com",
  "password": "P@ssw0rd!"
}

Responses

StatusDescription
200Returns AuthResponse with token and refresh token.
400Invalid credentials or account not confirmed.
POST /auth/register Public Register a new user

Description

Creates a new user account and sends an email confirmation code.

Request Body

{
  "userName": "johndoe",
  "email": "john@example.com",
  "discordHandle": "john#1234",
  "firstName": "John",
  "lastName": "Doe",
  "password": "P@ssw0rd!"
}

Responses

StatusDescription
200Registration successful. Confirmation email sent.
400Validation error (e.g., duplicate email, weak password).
POST /auth/refresh Public Refresh an expired JWT

Request Body

📖 Courses

GET /api/courses Public List courses (paginated)

Description

Returns a paginated list of course summaries. Anonymous access is allowed; authenticated users also get bookmark and enrollment status.

Query Parameters

NameTypeDefaultDescription
pageNumberint1Page number (≥ 1)
pageSizeint10Items per page (1–50)
searchValuestring?nullSearch by title
sortColumnstring?nullColumn to sort by
sortDirectionstring?ASCASC or DESC
isFreebool?nullFilter free / paid courses

Responses

StatusDescription
200Returns PaginatedList<CourseSummaryResponse>
GET /api/courses/{courseId}/content Public Get course content (sections & lessons)

Description

Returns the course's section and lesson hierarchy. Use the public HashId (e.g., Xy7zK), not the integer database ID.

Path Parameters

NameTypeDescription
courseIdstringHashed course ID

Responses

StatusDescription
200Returns CourseResponse (KeyId, Hours, Sections[])
404Course not found or invalid HashId.

POST /api/courses Auth Create a new course

Description

Creates a course and assigns the authenticated user as owner. Returns a Location header pointing to the new course's content endpoint.

Request Body

{
  "title": "Introduction to AI",
  "description": "Learn AI fundamentals...",
  "price": 0,
  "startin": "2025-09-01",
  "endin": "2025-12-01",
  "tags": [1, 3],
  "learningOutcomes": ["Understand neural networks"],
  "prerequisites": ["Basic Python"]
}

Responses

StatusDescription
201Course created. Location header included.
400Validation failed (invalid tags, dates, etc.).
PUT /api/courses/{courseId} UpdateCourses Update course details

Description

Updates the course's title, description, dates, tags, learning outcomes, and prerequisites. Requires UpdateCourses permission.

Request Body

{
  "title": "Advanced AI",
  "description": "Deep dive into...",
  "startin": "2025-09-01",
  "endin": "2026-01-01",
  "tags": [1, 5],
  "learningOutcomes": ["Master deep learning"],
  "prerequisites": ["Intro to AI course"]
}

Responses

StatusDescription
204Course updated.
404Course or tags not found.
PUT /api/courses/{courseId}/cover-image UpdateCourses Upload / replace cover image

Description

Uploads or replaces the cover image. Accepts multipart/form-data. The old image is deleted unless it's the default placeholder. Allowed formats: .jpg, .jpeg, .png. Max size: 10 MB.

Form Data

FieldTypeDescription
imagefileThe image file to upload

Responses

StatusDescription
204Image updated.
404Course not found.
POST /api/courses/{courseId}/enroll Auth Enroll in a course

Description

Enrolls the authenticated user. Idempotent — re-enrolls soft-deleted enrollments.

Responses

StatusDescription
204Enrolled successfully.
404Course not found.

Description

Soft-deletes the enrollment. Course owners cannot unenroll.

Responses

StatusDescription

404Not enrolled.

StatusDescription

🎬 Lessons

POST /api/lessons/init Auth Initialize lesson metadata

Description

Creates the lesson shell (step 1 of the two-step wizard). Returns the new lesson ID.

Request Body

{
  "title": "What is a Neural Network?",
  "sectionId": 5,
  "type": 1     // 1 = Video, 2 = Article, 3 = Quiz
}

Responses

StatusDescription
200Returns { lessonId: int }
400Validation error.
PUT /api/lessons/{id}/complete Auth Upload video & finalize lesson

Description

Step 2: uploads the video file and completes the lesson. Accepts multipart/form-data. Max upload: 1 GB.

Form Data

FieldTypeDescription
descriptionstring?Lesson description
isPreviewboolWhether the lesson is a free preview
scheduledDateDateTime?Scheduled publish date (must be future)
videoFilefile?The video file

Responses

StatusDescription
204Lesson completed.
404Lesson not found.
GET /api/lessons/{id}/stream Auth Stream lesson video

Description

Streams the lesson video file. Supports HTTP range requests for seeking. Response is not cached.

Responses

StatusDescription
200Video file stream (video/* content type).
404Lesson or video not found.

Description

Adds a review or updates an existing one. The user must be enrolled and cannot review their own course.
